## ⚠️ NOTES

### Automation & Testing

Please be aware that the code in this repository is deployed using a **fully automated process**. This automation pipeline **does not include any automated tests**.

### Non-stable API

This project relies on the **Cloud Development Kit for Terraform (CDKTF)**.

As highlighted in the [official HashiCorp documentation](https://developer.hashicorp.com/terraform/cdktf#:~:text=You%20are%20comfortable%20living%20on%20the%20cutting%20edge%3B%20CDKTF%20may%20still%20have%20breaking%20changes%20before%20our%201.0%20release.), CDKTF has not yet reached its 1.0 release. Please be aware that the API **may still be subject to breaking changes** until it is considered stable.
